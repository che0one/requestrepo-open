# Contributing to requestrepo

Thank you for considering contributing to requestrepo! We appreciate any help that you can provide.

## Getting Started

1. Fork the repository and clone it to your local machine.

2. Make sure that you have the required dependencies installed:

   - [Node.js](https://nodejs.org/) and [npm](https://www.npmjs.com/) (or [yarn](https://yarnpkg.com/)) for the frontend
   - [Python](https://www.python.org/) and [pip](https://pip.pypa.io/) for the backend
   - [Flask](https://flask.palletsprojects.com/) and [React](https://reactjs.org/) are used in the project

3. Run `npm install` (or `yarn install`) to install the frontend dependencies.

4. Run `pip install -r requirements.txt` to install the backend dependencies.

5. Create a new branch for your changes: `git checkout -b my-feature-branch`.

6. Make the necessary changes in the codebase.

7. Commit your changes and push them to your forked repository: `git push -u origin my-feature-branch`.

8. Create a new pull request from your forked repository to the main project's `main` branch.

## Contact

If you have any questions or doubts, feel free to contact the maintainers of this project by opening an issue.
